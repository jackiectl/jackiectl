## Research

<details>
<summary><b>LLM for Incentive Design in Mobility Systems</b> — <i>Can an LLM design the incentives that make shared mobility actually work?</i></summary>

An LLM-guided workflow that generates and evaluates interpretable incentive policies for ride-hailing, transit, and shared vehicles — pairing learning-based methods with classical optimization.

</details>

<details>
<summary><b>EPCOT Foundation Model — TSS-to-Expression Data Pipeline</b> — <i>Made a genomic foundation model 2–5× faster to train — and found a 50 kb bug that was quietly wrecking its interpretability.</i></summary>

Pseudobulk pipeline from raw snMultiome (1 bp sparse ATAC → 600 kb HDF5 windows; CP10K-normalized 19,264-gene RNA vectors across 7 cell types). Five bit-equivalent training optimizations for a 2–5× speedup at identical numerics. Reset-head fine-tuning for small-data generalization. Located and fixed a ~50 kb TSS-bin offset.

</details>

<details>
<summary><b>AI-Guided Environmental Sensing & Adaptive Monitoring</b> — <i>Where should you put the next sensor in Lake Michigan? Let uncertainty decide.</i></summary>

Reproducible ingestion → reconstruction → uncertainty-quantification workflow for the sparse, seasonal Great Lakes, comparing random / grid / uncertainty-aware sensor-placement rules.

</details>

<details>
<summary><b>Applets for Data Analytics in Materials Science</b> — <i>Turning materials-science analysis into something you can just... upload a CSV to.</i></summary>

JavaScript front end + Python back end serving the materials community: users upload datasets and analyze them with SOTA models and fitting methods, built as reusable FAIR-compliant modules.

</details>

<details>
<summary><b>Functional Genomic Representation Learning for Microbial Communities</b> — <i>A parameter-free way to turn a whole microbial community into one honest vector.</i></summary>

Proteogenic k-mer tokenization yields interpretable, parameter-free genome encodings that plug into both classical statistics and pretrained sequence models (Evo, ESM, ProtT5). Abundance-weighted community embeddings, leakage-safe features, permutation tests and honest cross-validation.

</details>
